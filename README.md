# ASTRO A50 Gen 4 PC Firmware Fix (2026)

This is the original setup and the last/latest firmware for the ASTRO A50 Gen 4 on PC, since Logitech no longer cares about it. You're welcome! :)

Dies ist das Original-Setup sowie die letzte aktuellste Firmware für das ASTRO A50 Gen 4 am PC, da Logitech sich nicht mehr darum kümmert. Bitteschön! :)

## 🇺🇸 Instructions (English)

1. Download **astro commandcenter.msi**.
2. Run and install the file.
3. Download the **mimas_...** file.
4. Start the Astro Command Center app.
5. Click on the gear icon in the top left corner.
6. Click on **"Update firmware manually"**.
7. Select the downloaded **mimas_...** file and let the update finish.

*Note: If you previously installed the version from the Microsoft Store, please uninstall it first!*

## 🇩🇪 Anleitung (Deutsch)

1. **astro commandcenter.msi** herunterladen.
2. Datei ausführen und installieren.
3. **mimas_...** Datei herunterladen.
4. Astro Command Center App starten.
5. Oben links auf das Zahnrad klicken.
6. Auf **"Firmware manuell aktualisieren"** klicken.
7. Die heruntergeladene **mimas_...** Datei auswählen und durchlaufen lassen.

*Hinweis: Falls du vorher die Version aus dem Microsoft Store installiert hattest, deinstalliere diese bitte zuerst!*
